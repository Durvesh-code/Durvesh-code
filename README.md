### Projects

**[HealthTrack+](https://github.com/Durvesh-code/HealthTrack-)**
Patient / doctor / pharmacy platform — Flask, MySQL/MongoDB, React 19. Includes a role-aware AI chatbot: SSE streaming, an Agno agent, 15 role-scoped tools across the three user types.

**[Advanced RAG System](https://github.com/Durvesh-code/Advanced_RAG_System)**
Hybrid retrieval (BM25 + dense via LangChain's EnsembleRetriever) with FlashRank cross-encoder reranking, served on FastAPI.

**[CreditWise](https://github.com/Durvesh-code/creditwise-loan-approval-system)**
Loan approval prediction — compared Logistic Regression, Naive Bayes, and KNN. Best model: 89% accuracy.

**[SmartCart](https://github.com/Durvesh-code/SmartCart-Clustering-system)**
Customer segmentation using unsupervised ML.

**[Evently](https://github.com/Durvesh-code/Evently)**
Event management platform — Java Spring Boot + React. Built during an Infosys Springboard internship.
